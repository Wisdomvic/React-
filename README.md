# 📝 React Blog List Application

A simple and dynamic blog listing application built with React.  
This project demonstrates core React concepts such as state management, props, conditional rendering, filtering, and deleting items dynamically.

---

## 🚀 Features

- Display a list of blog posts
- Filter blogs by author
- Delete a blog post dynamically
- Reusable BlogList component
- Conditional rendering of delete button
- State management using `useState`
- Dynamic rendering using `.map()`

---

## 🧠 Concepts Practiced

This project demonstrates:

- React functional components
- useState hook
- Passing data via props
- Passing functions as props
- Filtering arrays
- Updating state immutably
- Conditional rendering
- Handling click events

## 🔄 How It Works

- Blog data is stored in state inside `Home.js`.
- `BlogList` component receives blogs via props.
- The delete function is passed from parent to child.
- When delete is clicked:
  - The blog id is sent back to Home
  - The blog list is filtered
  - State updates
  - UI re-renders automatically
